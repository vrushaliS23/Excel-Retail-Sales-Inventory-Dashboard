# Retail Sales & Inventory Intelligence Dashboard

## Project Overview

This project is an interactive Excel dashboard designed to analyze retail sales and inventory performance. The dashboard provides insights into sales trends, profitability, inventory health, supplier contribution, customer performance, and stock turnover.

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Conditional Formatting
* Data Analysis

## Key KPIs

* Total Sales
* Total Profit
* Profit Margin %
* Total Orders
* Quantity Sold
* Inventory Value
* Inventory Turnover

## Dashboard Features

* Sales Trend Analysis
* Regional Performance Analysis
* Category-wise Sales & Profit
* Top Products by Profit
* Inventory Health Monitoring
* Low Stock Product Tracking
* Supplier Analysis
* Customer Analysis
* Interactive Slicers

## Business Insights

* Bangalore is the top-performing region.
* Fruit & Nut Chocolate is the most profitable product.
* ABC Traders contributes the highest inventory value.
* Profit Margin is 46%.
* Fast-moving products require regular replenishment.
* Low-turnover products need promotional strategies.
